INSTALLATION:

1. Locate the template directory in your Phorum installation (such as YourPhorumDirectory/templates).
Upload the bitmapworld folder and all contents to the templates directory.

2. Go to your Admin control panel. The new template should appear on your list of available templates. Simply activate.

3. If you are using the "Announcements" mod, then get the bitmapworld folder from mods-announcements-templates and upload it to
YourPhorumDirectory/mods/announcements/templates


SIDEBAR:

The text in the sidebar is found in a file called sidebarlinks.tpl. This file can contain HTML or Phorum template tags.
